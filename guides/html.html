HTML: The Structure of the Web
HTML stands for HyperText Markup Language. It gives structure to the content on the web.

Basic HTML Structure


<!DOCTYPE html>
<html>
<head>
    <title>Title of the document</title>
</head>
<body>
    The content of the document......
</body>
</html>

Core HTML Elements

HTML documents are formed using elements. Elements are denoted by tags. For instance:

  • <h1> to <h6>: These tags render headings, with <h1> being the highest level and <h6> the lowest.
  • <p>: Represents a paragraph.
  • <a href="url">Link Text</a>: Creates a hyperlink to other web pages or resources. The "href" attribute holds the link's destination address.
  • <img src="image_path.jpg" alt="Description">: Displays an image. The "src" attribute holds the image path. The "alt" attribute provides a text description of the image, which improves accessibility.
  • List Elements: <ul> (unordered/bullet list), <ol> (ordered/numbered list), and <li> (list item).
  • <div>: A container that groups other elements together. It's a block-level element.
  • <span>: Used to group inline-elements or single characters to apply CSS or JavaScript to them.

Forms & Input

Forms collect user input. The <form> element wraps input elements like <input>, <textarea>, <select>, etc. 


<form action="/submit_form" method="post">
    <input type="text" name="name" placeholder="Enter your name">
    <input type="submit" value="Submit">
</form>
CSS: Styling the Web
CSS stands for Cascading Style Sheets. It provides the look and feel of the web pages.

Basic CSS Styling


body {
    font-family: Arial, sans-serif;
    background-color: #e0e0e0;
}
h1 {
    color: #333;
}

The CSS Box Model

Every element in CSS is treated as a rectangular box, consisting of padding, border, margin, and the actual content. 


div {
    width: 300px;
    padding: 10px;
    border: 5px solid black;
    margin: 20px;
}

CSS Flexbox

Flexbox is a CSS layout model that allows items in a container to be dynamically arranged based on certain properties.


.flex-container {
    display: flex;
    justify-content: space-between;
}

CSS Grid

Grid is a 2D layout system, allowing you to build complex web design layouts with ease.


.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
}
JavaScript: Making the Web Interactive
JavaScript enables interactivity and dynamic content on the web.

Basic JavaScript


let greeting = "Hello, Web!";
alert(greeting);

Variables, Functions, and Loops

JavaScript has several core concepts including variables (store data), functions (perform actions), and loops (repeat actions). 


let age = 25;
function greet(name) {
    return "Hello, " + name + "!";
}
for (let i = 0; i < 5; i++) {
    console.log(i);
}

Advanced JavaScript

As you dive deeper, concepts like promises, async/await, arrow functions, and more become vital. 


const fetchData = async () => {
    let response = await fetch('https://api.example.com/data');
    let data = await response.json();
    return data;
}
fetchData().then(data => console.log(data));
