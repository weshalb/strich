Countdown Timer

By: wovor

Simple countdown timer in js.

let timeLeft = 10;  // in seconds const timerElement = document.getElementById('timer');  function startCountdown() {     setInterval(() => {         if (timeLeft > 0) {             timeLeft--;             timerElement.textContent = `Time left: ${timeLeft} seconds`;         } else {             timerElement.textContent = "Time's up!";         }     }, 1000); }
