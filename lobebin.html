Ultimate Web Development Guide

HTML: HyperText Markup Language

HTML (HyperText Markup Language) is the backbone of web content. It describes the structure of a web page.

Basic HTML Structure

Standard HTML Page Structure

<!DOCTYPE html>
<html>
<head>
    <title>Title of the document</title>
</head>
<body>
    The content of the document...
</body>
</html>

HTML Elements

Heading
<h1>This is a heading</h1>
Paragraph
<p>This is a paragraph.</p>

HTML Attributes

Attributes provide additional information about elements and are always specified in the start tag.

Anchor Element with href Attribute
<a href="https://www.example.com">This is a link</a>

CSS: Cascading Style Sheets

CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS Selectors

Styling a Paragraph Element
p {
    color: red;
}

CSS Properties

Setting Background and Font Size
body {
    background-color: #f0f0f0;
    font-size: 16px;
}

Variables

Declaring and Initializing a Variable
let x = 5;

A Simple Function to Add Two Numbers
